Module mfwk.path

Convenience functions for paths.

Functions

Combine(Variable)
Concatenates arguments into a file path. E.g. mfwk.path.Combine( "foo", "bar", "module", ".lua" ) will return foo/bar/module.lua.

Parameters:

  • Variable {string,...} amount of path parts.

Returns:

    string Concatenated path.
Directory(path)
Returns the sub-directory of path. E.g. mfwk.path.Directory( "foo/bar/module.lua" ) will return foo/bar.

Parameters:

  • path string Path to get the sub-directory of.

Returns:

    string Sub-directory of path.
File(path)
Returns the file name and extension of path. E.g. mfwk.path.File( "foo/bar/module.lua" ) will return module.lua.

Parameters:

  • path string Path to get the file name and extension of.

Returns:

    string File name and extension after the last path separator.
FileWithoutExtension(path)
Returns the file name of path, without file extension. E.g. mfwk.path.FileWithoutExtension( "foo/bar/module.lua" ) will return module.

Parameters:

  • path string Path to get the file name of.

Returns:

    string File name after the last path separator, without extension.
WithoutExtension(path)
Trims trailing extension from path, if it has one. E.g. mfwk.path.WithoutExtension( "foo/bar/module.lua" ) will return foo/bar/module.

Parameters:

  • path string Path to trim.

Returns:

    string Path without trailing extension.
