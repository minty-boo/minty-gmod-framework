Filter(tbl, filter)
Filters tbl, given a filter function. Assumes tbl is numerically and sequentially indexed.

Parameters:

  • tbl table Table to filter.
  • filter func Filter function, e.g. function( v ) return ( type( v ) == "string" ) end to filter out all strings.

Returns:

    table Filtered results.
FindName(tbl)
Finds the fully qualified name of tbl. E.g. local tbl = mfwk.table; mfwk.table.FindName( tbl ) would output "mfwk.table".

Parameters:

  • tbl table Table to find the name of.

Returns:

    string Name of tbl, or nil if it could not be found.
FindParent(tbl)
Finds the root table containing tbl. E.g. local tbl = mfwk.table; mfwk.table.FindParent( tbl ) would output mfwk.

Parameters:

  • tbl table Table to find the parent table of.

Returns:

    table Parent table, or nil if it could not be found.
Remove(tbl, kv)
Removes an entry from tbl, by number index or value. Assumes tbl is numerically and sequentially indexed.

Parameters:

  • tbl table Table to remove from.
  • kv Number index, or value.

Returns:

    Removed value, or nil if the index was invalid/value could not be found.
Sort(tbl, sorter)
Sorts tbl, given a sorter function. Assumes tbl is numerically and sequentially indexed.

Parameters:

  • tbl table Table to sort.
  • sorter func Sorting function, e.g. function( a, b ) return ( #a < #b ) end to sort by length (ascending).

Returns:

    table Sorted results.
